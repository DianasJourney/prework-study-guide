HTML

  • Enter your HTML notes here
  • The body element represents the visible content shown to the user.
  • The body element is also called the parent element which includes children elements.
  • HTML is the structure for the website.
  • The head element contains information, or metadata that helps the browser render the page correctly. Elements included within the head are not visible to users who look at the webpage.
  • Body element contains the content that is shown to the user. All the elements that we want the user to see and interact with are placed in this element.
  • HTML elements are always defined in the beginning element and never the closing element.
  • The base element defines the base url for a webpage.
  • The link element connects the external resource to the HTML document.
  • The meta element defines metadata such as character set, description, keywords, author, and viewport.

CSS

  • Enter your CSS notes here
  • The CSS style sheet must be linked to the HTML file before it can render its properties by using the link element.
  • CSS is where we design what we want inside our structure.
  • Margin and Padding are not the same. Padding property adds space around the content inside an element. When increasing padding, we are actually making an entire element bigger.
  • Margin property acts much like a margin in a word processing document. A marin property indicates how much space we want around the outside of the element.
  • Margin and Padding are not visible on the page but they add a buffer.
  • Essentially HTML element is essentially a box. Each box consists of margin outside the element, and the element's border, padding, and content.
